# 医美智能营销管理系统使用手册

## 系统概述

本系统是一套完整的医美诊所智能营销管理解决方案，集成了 AI 客服、客户画像分析、内容营销、小红书运营等核心功能，帮助医美诊所提升营销效率和客户转化率。

---

## 核心功能模块

### 1. AI 智能客服（/chat）

**功能特点：**
- 24小时在线接待客户咨询
- 自动检索知识库提供专业回答
- 自动提取客户信息（姓名、手机、微信、兴趣项目）
- 对话自动同步到 Airtable CRM
- 基于客户历史记录提供个性化服务
- 自动识别客户心理类型（恐惧型/贪婪型/安全型/敏感型）

**使用流程：**
1. 客户访问 `/chat` 页面开始对话
2. AI 自动检索相关知识库内容回答问题
3. 当客户提供手机号时，自动创建线索记录
4. 系统自动分析对话识别客户心理类型
5. 对话记录自动同步到 Airtable（如已配置）

**配置要点：**
- 在知识库管理中维护常见问题和回答
- 定期更新项目信息和价格
- 在系统配置中填写 Airtable API 信息

### 2. 统一管理后台（/dashboard）

#### 2.1 系统配置（/dashboard/config）

**功能：**
- Airtable API 配置
- 系统基础设置

**配置步骤：**
1. 登录 Airtable，创建 Base 和 Table
2. 获取 API Token（Settings → Developer Hub → Personal Access Tokens）
3. 复制 Base ID（Base URL 中的 app 开头的字符串）
4. 在系统配置页面填写并保存

#### 2.2 数据分析（/dashboard/analytics）

**功能：**
- 实时访问统计（PV/UV）
- 对话转化率分析
- 线索来源分析
- 客户画像分布

**关键指标：**
- **总访问量**：网站总访问次数
- **独立访客**：去重后的访客数量
- **对话数**：AI 客服对话总数
- **线索数**：成功提取客户信息的数量
- **转化率**：线索数 / 对话数

#### 2.3 对话管理（/dashboard/conversations）

**功能：**
- 查看所有对话记录
- 按状态筛选（进行中/已结束）
- 按时间范围筛选
- 查看对话详情和客户信息

**使用技巧：**
- 定期查看对话记录，了解客户常见问题
- 关注未成功提取信息的对话，优化话术
- 分析高转化对话的特点

#### 2.4 知识库管理（/dashboard/knowledge）

**功能：**
- 管理客户问询知识库（C端）
- 管理内部管理知识库（B端）
- 知识库分类和标签
- 知识库搜索和筛选
- 使用统计

**知识库类型：**

**客户问询知识库（对外）：**
- 项目介绍（超皮秒、水光针、热玛吉、牙齿美白等）
- 常见问题（效果、价格、恢复期、风险等）
- 注意事项（术前术后护理）
- 价格政策

**内部管理知识库（对内）：**
- 销售话术（开场白、需求挖掘、促成技巧）
- 心理分析（客户决策类型、消费动机）
- 异议处理（价格异议、效果疑虑、安全顾虑）
- 成交技巧

**维护建议：**
- 每周更新项目信息和价格
- 根据客户常见问题补充知识库
- 定期review使用统计，优化高频知识

#### 2.5 内容管理（/dashboard/content）

**功能：**
- 一键生成小红书爽文
- 支持多种内容类型
- 支持多种语气风格
- 自动生成配图
- 一键复制到剪贴板

**内容类型：**
- 项目体验分享
- 效果对比展示
- 价格揭秘
- 避坑指南
- 节日营销

**语气风格：**
- 热情洋溢
- 专业严谨
- 轻松随意

**图片风格：**
- 现代简约
- 优雅高级
- 活力鲜艳

**使用流程：**
1. 选择内容类型和项目
2. 设置语气风格
3. 添加关键词（可选）
4. 点击"一键生成爽文"
5. 查看生成结果
6. 点击"生成配图"生成图片
7. 复制内容到小红书发布

#### 2.6 客户管理（/dashboard/customers）

**功能：**
- 查看所有客户（100位测试数据）
- 多维度筛选（客户分层、心理类型、消费能力）
- 客户详情查看
- 客户画像分析

**客户分层：**
- **A级客户**：高消费能力 + 积极心理类型（贪婪型/敏感型）
- **B级客户**：中等消费能力或混合心理类型
- **C级客户**：低消费能力或消极心理类型
- **D级客户**：低消费能力 + 消极心理类型

**心理类型：**
- **恐惧型**：怕老、怕丑、怕被比下去
- **贪婪型**：追求极致效果、性价比
- **安全型**：注重风险、要权威认证
- **敏感型**：在意他人评价、社交压力

**使用技巧：**
- 针对不同心理类型使用不同话术
- 优先跟进 A/B 级客户
- 定期分析客户画像分布，优化营销策略

#### 2.7 小红书运营（/dashboard/xiaohongshu）

**功能：**
- 内容发布管理
- 数据监控（阅读量、点赞、评论、分享、收藏）
- 内容筛选（已发布/待发布/草稿）
- 爆款内容分析

**使用流程：**
1. 在内容管理中生成小红书文案和配图
2. 在小红书运营页面创建新内容
3. 填写标题、正文、标签
4. 设置发布时间（立即发布或定时发布）
5. 发布后定期更新数据（手动输入阅读量、点赞数等）
6. 分析爆款内容特点

## 数据库表结构

### 核心表

**user（用户表）**
- 存储系统用户信息
- 支持角色管理（admin/user）

**leads（线索表）**
- 存储客户线索信息
- 包含客户画像字段（心理类型、消费能力、客户分层）

**conversations（对话表）**
- 存储 AI 客服对话记录
- 包含心理标签识别结果

**messages（消息表）**
- 存储对话中的每条消息

**knowledge_base（知识库表）**
- 存储知识库内容
- 支持分类和标签
- 区分客户问询和内部管理两种类型

**xiaohongshu_posts（小红书内容表）**
- 存储小红书发布内容
- 包含数据统计字段

**xiaohongshu_comments（小红书评论表）**
- 存储小红书评论记录

**automation_triggers（自动化触发器表）**
- 存储自动化营销规则（预留）

## API 接口文档

### tRPC Router 结构

```typescript
appRouter = {
  auth: {
    me: () => User,           // 获取当前用户
    logout: () => boolean,    // 登出
  },
  chat: {
    sendMessage: (params) => Message,      // 发送消息
    getConversations: () => Conversation[], // 获取对话列表
    getMessages: (conversationId) => Message[], // 获取消息列表
  },
  admin: {
    getConfig: () => Config,               // 获取配置
    updateConfig: (config) => boolean,     // 更新配置
  },
  analytics: {
    getOverview: () => Stats,              // 获取概览数据
  },
  knowledge: {
    list: (params) => Knowledge[],         // 获取知识库列表
    create: (data) => Knowledge,           // 创建知识库
    update: (id, data) => Knowledge,       // 更新知识库
    delete: (id) => boolean,               // 删除知识库
  },
  content: {
    generate: (params) => Content,         // 生成内容
    generateImage: (params) => Image,      // 生成图片
  },
  customers: {
    list: (params) => Customer[],          // 获取客户列表
    get: (id) => Customer,                 // 获取客户详情
  },
  xiaohongshu: {
    getStats: () => Stats,                 // 获取统计数据
    getPosts: (params) => Post[],          // 获取内容列表
    createPost: (data) => Post,            // 创建内容
    updatePost: (id, data) => Post,        // 更新内容
  },
}
```

## 部署和运维

### 环境变量

系统自动注入以下环境变量，无需手动配置：

```
DATABASE_URL=<MySQL连接字符串>
JWT_SECRET=<JWT密钥>
VITE_APP_ID=<应用ID>
OAUTH_SERVER_URL=<OAuth服务器地址>
VITE_OAUTH_PORTAL_URL=<OAuth登录页面>
BUILT_IN_FORGE_API_URL=<内置API地址>
BUILT_IN_FORGE_API_KEY=<内置API密钥>
```

### 数据备份

**重要提示：** 定期备份数据库，避免数据丢失。

```bash
# 导出数据库
mysqldump -u username -p database_name > backup.sql

# 导入数据库
mysql -u username -p database_name < backup.sql
```

2. **知识库缓存**：知识库查询结果会自动缓存

## 常见问题

### Q1: AI 客服回答不准确怎么办？

**A:** 
1. 检查知识库内容是否完整
2. 确保知识库状态为"激活"
3. 优化知识库标题和标签，提高匹配准确度

### Q2: Airtable 同步失败怎么办？

**A:**
1. 检查 API Token 是否正确
2. 检查 Base ID 是否正确
3. 确保 Airtable 表结构包含必要字段

### Q3: 图片生成失败怎么办？

**A:**
1. 检查网络连接
2. 稍后重试
3. 尝试更换图片风格

### Q4: 客户心理类型识别不准确怎么办？

**A:**
1. 需要多轮对话才能准确识别
2. 可以在客户详情中手动修改
3. 定期review识别结果，优化识别算法

## 附录

### A. 客户心理类型详解

**恐惧型客户：**
- **特征**：担心变老、变丑，害怕被比下去
- **话术**：强调效果持久性、安全性、成功案例
- **案例**："您看这位客户，做完超皮秒后斑点几乎看不见了，而且恢复特别快..."

**贪婪型客户：**
- **特征**：追求极致效果和性价比
- **话术**：强调性价比、优惠活动、组合套餐
- **案例**："现在做超皮秒+水光针套餐，比单独做能省2000元..."

**安全型客户：**
- **特征**：注重风险、要权威认证
- **话术**：强调资质、设备、医生经验、安全保障
- **案例**："我们使用的是美国FDA认证的超皮秒设备，医生有10年以上临床经验..."

**敏感型客户：**
- **特征**：在意他人评价、社交压力
- **话术**：强调自然效果、不被发现、提升自信
- **案例**："做完超皮秒后，皮肤会自然变白，别人只会觉得你最近气色好..."

### B. 小红书爆款内容特征

1. **标题吸引眼球**：使用数字、emoji、制造悬念
2. **真实接地气**：第一人称叙述，分享真实感受
3. **结构清晰**：分点、分段，易于阅读
4. **视觉丰富**：配图精美，emoji使用恰当
5. **引导互动**：结尾引导评论、私信
6. **话题标签**：使用热门话题标签

### C. 数据分析指标

**访问指标：**
- PV（Page View）：页面浏览量
- UV（Unique Visitor）：独立访客数
- 跳出率：只访问一个页面就离开的比例

**转化指标：**
- 对话率：对话数 / UV
- 线索转化率：线索数 / 对话数
- 预约转化率：预约数 / 线索数

**内容指标：**
- 阅读量：小红书内容阅读次数
- 互动率：（点赞+评论+收藏）/ 阅读量
- 转化率：私信咨询数 / 阅读量
