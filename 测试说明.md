## 🚀 快速配置数据库

1. 创建 `.env` 文件：
   ```bash
   cp .env.example .env
   ```

2. 编辑 `.env`，设置数据库：
   ```env
   DATABASE_URL=mysql://root:password@localhost:3306/medical_beauty_crm
   ```

3. 创建数据库和表：
   ```bash
   mysql -u root -p -e "CREATE DATABASE medical_beauty_crm;"
   npm run db:push
   ```

4. 重启服务器或刷新页面
